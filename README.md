# Cách setup và chạy

## 1. Clone project về máy

```
git clone git@github.com:DangNH1652005/Puff_Bear.git
```

## 2. Vào project

```
cd FER_202
```

## 3. Cài packages

```
npm install
```

## 4. Chạy project

```
npm run dev
```
